## 📌 О проекте
Этот репозиторий создан как дополнение к выпускной квалификационной работе (ВКР), посвящённой разработке прототипа аналитического продукта для мониторинга рынка IT-труда в российском банковском секторе. Код разбит на тематические Jupyter Notebook, реализующие полный цикл сбора, обработки и анализа данных из открытых источников (OSINT). Основная цель — предоставить HR-департаментам банков инструмент для оценки внешней среды: конкурентных зарплат, востребованных навыков, удовлетворённости сотрудников и репутации работодателя.

## 🎯 Цели и задачи исследования

**Цель работы** – разработать функциональный прототип дашборда, который на основе автоматизированного сбора и анализа открытых данных предоставляет количественные и качественные метрики о рынке IT-специалистов в российском банкинге.

**Задачи исследования:**
- Провести анализ существующих HR-инструментов и обосновать необходимость внешней аналитики.
- Реализовать систему сбора данных из открытых источников: отзывов сотрудников (dreamjob.ru), вакансий и резюме (HeadHunter).
- Разработать NLP-модели для анализа тональности текстов отзывов и автоматического выделения ключевых тем.
- Сгенерировать синтетические данные для прототипирования в условиях ограниченного доступа к API HeadHunter.
- Определить систему HR-метрик (eNPS, индекс удовлетворённости по факторам FSI, композитный индекс HHBI, индекс конкуренции и др.) и визуализировать их в дашборде.
- Оценить экономическую эффективность продукта через две финансовые модели (внутренний продукт для банка и коммерческий SaaS).

## 📂 Файловая структура

### Ноутбуки для сбора и обработки данных

- `dreamjob_parser.ipynb` – парсер отзывов сотрудников с сайта dreamjob.ru для девяти крупнейших российских банков. Извлекает должность, город, дату, оценку, тексты плюсов и минусов, ответ компании.
- `parser_hh_vacancies.ipynb` – сбор вакансий через официальное API HeadHunter с фильтрацией по IT и финансовому сектору. Сохраняет структурированную информацию: зарплатные вилки, требования, регион, работодатель.
- `resume_hh_parsing.ipynb` – комбинированный парсер резюме с HeadHunter. Использует Selenium для обхода поисковой выдачи и асинхронные HTTP-запросы (`httpx`) для загрузки и парсинга детальных страниц резюме. Извлекает до 30 полей: желаемая должность, опыт, навыки, образование, зарплатные ожидания и др.
- `generated_datasets.ipynb` – генератор синтетических датасетов вакансий и резюме. Поскольку бесплатный API HeadHunter не предоставляет репрезентативной выборки, данные созданы на основе экспертных правил и вероятностных распределений (возраст, опыт, зарплата, география) для построения прототипа дашборда.
- `reviews_prepare_calc_metrics.ipynb` – основной ноутбук для анализа отзывов. Включает предобработку текста (очистка, лемматизация с `pymorphy3`), классификацию тональности и выделение ключевых тем с помощью большой языковой модели (GPT-4o-mini), расчёт HR-метрик (eNPS, FSI, HHBI) и агрегацию по банкам.

### Примеры данных

- `vacancies_ready_.xlsx` – пример результирующего файла с обработанными и агрегированными данными по вакансиям (может содержать как реальные, так и синтетические данные для демонстрации).
- [**Спарсенные данные резюме с HeadHunter**](https://docs.google.com/spreadsheets/d/1GNDbKQfC3OVZBv9284d_jiB8VXI0Usz7/edit?usp=sharing&ouid=100639999688557151332&rtpof=true&sd=true) – ссылка на Google Sheets с реальными данными резюме, собранными в ходе тестирования парсера.

## 📊 Результаты

На основе обработанных данных построен интерактивный прототип дашборда в **Яндекс DataLens**, доступный по ссылке:  
🔗 [https://datalens.yandex/utr0jy4f5e24e](https://datalens.yandex/utr0jy4f5e24e)

Дашборд включает две вкладки:
1. **Спрос и предложение** – индекс конкуренции, сравнение зарплатных ожиданий, география вакансий, среднее время перехода между работами.
2. **Аналитика отзывов** – общая статистика, распределение композитного индекса HHBI, динамика eNPS и рейтингов по банкам.

## 📥 Источники данных

- Отзывы сотрудников: [dreamjob.ru](https://dreamjob.ru)
- Вакансии и резюме: официальное API и веб-интерфейс [HeadHunter](https://hh.ru)

## 📌 Примечание

В связи с ограничениями бесплатного доступа к API HeadHunter часть данных для прототипа была сгенерирована синтетически. Все скрипты сбора реальных данных также представлены в репозитории и могут быть использованы при наличии платного доступа или легальных методов парсинга.
